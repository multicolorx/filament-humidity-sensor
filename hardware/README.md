# Hardware

This directory contains the PCB design files, release gerbers, and supporting component datasheets for the filament humidity sensor.

## Quick Links

- [Schematic source](kicad/filament-humidity-sensor.kicad_sch)
- [PCB layout source](kicad/filament-humidity-sensor.kicad_pcb)
- [KiCad project](kicad/filament-humidity-sensor.kicad_pro)
- [Gerber and drill files](gerbers/rev001/)
- [Zipped Gerber release](gerbers/gerbers-rev001.zip)
- [Component datasheets](datasheets/)

## Layout

- `kicad/` editable KiCad project files
- `gerbers/` manufacturing outputs
- `datasheets/` PDFs and text notes for selected components
- `backups/` KiCad backup archives

Open the KiCad project with KiCad 9 or newer.
